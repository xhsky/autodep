# 更新



## 代码更新
- 代码更新分为前端和后端两种方式



### 前端代码打包方式

- 说明
  - 将前端代码目录打包为`tar.gz`格式的压缩包
  - 该压缩包内只存在一个目录, 即代码目录, 其目录下是代码文件

- 压缩包组织格式示例
  ```
  - code.tar.gz
    - code
      - a.html
      - b.html
      - dir1
        c.html
  ```



### 后端代码打包方式

- 说明

  - 将后端`jar`包打包为`tar.gz`格式的压缩包
  - 该压缩包内只存在一个目录, 即代码木兰路, 其目录下是单一的`jar`包

- 压缩包组织格式示例

  ```
  - code.tar.gz
    - code
      - code.jar
  ```
